Automated CI/CD Pipeline Using Jenkins
======================================

Project Description
-------------------

This project demonstrates the implementation of a Continuous Integration and Continuous Deployment (CI/CD) pipeline using Jenkins on an AWS EC2 instance. Jenkins was configured to integrate with a GitHub repository and execute pipeline stages defined in a Jenkinsfile.

Technologies Used
-----------------

- AWS EC2
- Jenkins
- GitHub
- Git
- Linux

Project Workflow
----------------

- Launched an AWS EC2 instance.
- Installed and configured Jenkins.
- Created a GitHub repository.
- Added a Jenkinsfile to define the pipeline.
- Connected Jenkins with GitHub.
- Created a Jenkins Pipeline job.
- Executed the pipeline successfully.
- Verified build status and console output.

Features
--------

- Automated build execution using Jenkins.
- Integration with GitHub repository.
- Pipeline as Code using Jenkinsfile.
- Build monitoring through Jenkins Dashboard.
- Console output verification.
